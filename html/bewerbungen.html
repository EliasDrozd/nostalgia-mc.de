Bewerbung als Developer


Was erwartet dich?

Als Developer auf unserem Server ist es deine Aufgabe, Plugins zu Entwickeln, die bespielsweise für Special Events, Events oder Optimierungen gedacht sind.



Was sind die Anforderungen?

Du solltest Grundwissen in den folgenden Programmiersprachen aufweisen können:

  • Java
  • JavaScript
  • HTML
  • CSS
  • Bootstrap


    • Optimal zum Entwickeln von Minecraft Plugins wäre es von Vorteil, wenn du zumindest die Grundkenntnisse in den folgenden Bibliotheken hast:

  • Bukkit
  • Paper
  • Spigot
